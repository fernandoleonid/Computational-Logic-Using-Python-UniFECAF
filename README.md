# Computational Logic Using Python — UniFECAF

Repositório de apoio às aulas gravadas da disciplina **Computational Logic Using Python**, ministrada pelo **Prof. Fernando Leonid** na **UniFECAF**.

---

## Para que serve este repositório?

Este repositório é um **recurso de apoio**, não um atalho.

Se durante as aulas você escreveu seu código junto com o professor e ele não executou, ou apareceu um erro que você não conseguiu resolver, aqui você pode consultar o código de referência para identificar onde algo saiu diferente.

---

## Como usar

1. Acompanhe as aulas e **escreva seu próprio código** junto com o professor.
2. Se seu código não funcionar e você já tentou resolver o erro, consulte o arquivo correspondente neste repositório.
3. Compare, entenda a diferença e corrija o seu.

> **Importante:** consultar o código aqui antes de tentar por conta própria vai fazer você perder a oportunidade de aprender de verdade.

---

## A lógica do aprendizado

Será que aprendemos a andar de bicicleta sem nunca ter tentado pedalar?  
Será que andamos de skate sem nunca ter caído?  
Será que aprendemos a surfar sem nunca ter pegado uma onda?

**Só aprendemos fazendo e errando.** Errar faz parte do processo — é exatamente ali que o aprendizado acontece.
